# **App Name**: WaChat

## Core Features:

- Email/Password Authentication: Securely authenticate users via email and password.
- Email Verification: Verify user emails to activate accounts.
- OAuth Authentication: Enable user authentication via Google and Apple OAuth.
- WaChat AI: Chat bot just like meta ai with websearch enabled.
- Custom AI: User can create and custom ai.
- Multi-Device Linking: Link multiple devices to a single account, allowing users to access WaChat on different platforms, using a secure tool for transferring session keys.
- Real-time Messaging: Send and receive real-time messages between users.
- End-to-End Encryption (E2EE): Secure messages with end-to-end encryption to ensure privacy.
- Media Sharing: Share media files (images, videos, voice notes) securely.

## Style Guidelines:

- Primary color: Deep blue (#2E3192) to evoke security and trust.
- Background color: Light gray (#E5E7EB), a desaturated form of the primary to reduce eye strain.
- Accent color: Turquoise (#30D5C8), a brighter analogous hue for interactive elements, call-to-action buttons and highlights.
- Font: 'Inter', a grotesque-style sans-serif, will be used for headlines and body text due to its modern and neutral look.
- Use minimalist, consistent icons to represent different features.
- Maintain a clean and intuitive layout with clear visual hierarchy.
- Subtle animations for loading indicators and transitions to enhance user experience.
